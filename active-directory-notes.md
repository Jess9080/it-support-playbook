# Active Directory - Notas básicas

## Gestión de usuarios

- Creación de usuarios
- Eliminación de usuarios
- Restablecimiento de contraseña
- Desbloqueo de cuentas

## Gestión de grupos

- Añadir usuarios a grupos
- Control de permisos

## Incidencias comunes

- Usuario bloqueado
- Contraseña expirada
- Falta de permisos

## Buenas prácticas

- Asignar permisos por grupos
- Mantener estructura organizada
- Documentar cambios
