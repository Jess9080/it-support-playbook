# Incidencias comunes en soporte IT

## 1. Usuario no puede iniciar sesión

### Posibles causas:
- Contraseña incorrecta
- Cuenta bloqueada
- Problemas con Active Directory

### Solución:
- Verificar credenciales
- Desbloquear usuario en AD
- Restablecer contraseña
