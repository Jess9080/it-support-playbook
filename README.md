# IT Support Playbook

Repositorio con documentación práctica orientada a soporte IT, resolución de incidencias y administración básica de sistemas.

## Áreas cubiertas

- Gestión de incidencias (nivel 1 y 2)
- Diagnóstico de problemas
- Administración de usuarios
- Active Directory (conceptos básicos)
- Redes (IP, conectividad, diagnóstico)
- Procedimientos técnicos

## Objetivo

Servir como guía práctica para entornos de soporte IT, facilitando la resolución de problemas y la gestión eficiente de sistemas.

## Contenido inicial

- Procedimientos básicos de diagnóstico
- Incidencias comunes y soluciones
- Comandos útiles para soporte técnico

## Tecnologías relacionadas

- Windows / Windows Server
- Linux
- Redes (TCP/IP)
- Active Directory
